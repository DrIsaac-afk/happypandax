Plugin API¶

Interface¶

The plugin interface module named __hpx__. Import it in your plugin to access the methods and classes defined by the module:

import __hpx__ as hpx
logger = hpx.get_logger(__name__)

In addition to what is defined here, exceptions defined by HPX are also available at the module-level and can be imported like so:

import __hpx__ as hpx
from __hpx__ import PluginError

raise PluginError("", "")
raise hpx.PluginError("", "")
